# 📘 Mid-Semester Coding Assignment

### JavaScript & React Fundamentals

---

## 🎯 Objective

Develop a small web-based application (or a set of scripts) that demonstrates core concepts of:

* JavaScript ES6
* JSON handling
* Asynchronous programming
* Fetch API
* Basic React state management

This assignment is designed to help you prepare for the **mid-term exam**.

---

## 📋 Requirements

Your implementation must include the following components:

---

### 1️⃣ ES6 Features & Classes

* Demonstrate the difference between `var`, `let`, and `const` (scope + reassignment)
* Use `let` and `const` appropriately in your code
* Create an **arrow function** to calculate the square of a number
* Implement a class `Student`:

  * Properties: `name`, `marks`
  * Method: `display()` to print student details

---

### 2️⃣ JSON Data Handling

* Create a JavaScript object with:

  * `name`, `age`, `course`
* Convert the object into JSON using `JSON.stringify()`
* Convert JSON back to an object using `JSON.parse()`
* Display results in console or UI

---

### 3️⃣ Fetch API – Data Retrieval

* Fetch data from:

  ```
  https://jsonplaceholder.typicode.com/posts
  ```
* Display **titles of the first 5 posts**

---

### 4️⃣ Asynchronous Programming & Error Handling

* Fetch user data from a public API
* Display the retrieved data
* Handle errors using `.catch()`
* Demonstrate:

  * ✅ Successful response
  * ❌ Failed request (e.g., incorrect URL)

---

### 5️⃣ React State Management

Create a React component that:

* Uses the `useState` hook
* Displays a counter value
* Increments the counter on button click

---

## 🚀 Goal

By completing this assignment, you will:

* Strengthen your understanding of **ES6 concepts**
* Learn how to handle **JSON data effectively**
* Practice **API integration using Fetch**
* Understand **asynchronous JavaScript and error handling**
* Build confidence in **React state management**

---

## 📦 Submission Requirements

* ✅ Complete working code
* ✅ Clear and concise explanations for each section
* 🖼️ (Optional) Screenshots of output or UI

---

## 💡 Notes

* Use **modern JavaScript (ES6+)**
* Keep your code clean and well-commented
* Focus on concepts — this is **mid-term preparation**
